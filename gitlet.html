Gitlet

March 2015

Overview

Language Used: Java

For CS 61B (Data Structures) I implemented a version control system that mimics some of the basic features of the popular version control system git. This was made possible through the usage of Java's Serializable interface, which writes objects to a file before terminating the program.

Check out the original project guidelines here

Cool Features

One of Gitlet's most challenging features to build was rebase, a command that finds the split point of the current branch and the given branch, then snaps off the current branch at this point, then reattaches the current branch to the head of the given branch.
